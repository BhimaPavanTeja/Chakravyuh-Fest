# **App Name**: Chakravyuh Fest

## Core Features:

- Countdown Timer: Display a countdown timer on the homepage leading up to March 7th, 2026.
- About Fest Section: Information about the fest including theme, date & venue, organizing committee, and a message from the chairperson.
- Events Showcase: Display the event with categories: Technical, Non-Technical, and Sports.
- Schedule Overview: Present the fest schedule, highlighting Day 1 and Day 2 timelines.
- Gallery Display: Showcase photos, previous fest highlights, and videos in an organized gallery.
- Team Introduction: Introduce the organizing team including the Chairperson, Vice Chairs, and Event Heads.
- Sponsors Recognition: A section to display and acknowledge the fest's sponsors (Title Sponsor, Co-Sponsors, Partners).

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to evoke a sense of mystery and academic prestige, nodding to the challenges of 'Chakravyuh'.
- Background color: Very light gray (#F0F0F0), close in hue to indigo but heavily desaturated, for a modern, clean look.
- Accent color: Vivid purple (#BE90D4), a close analogue to indigo but brighter, to draw the eye.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text to provide a modern, tech-forward aesthetic.
- Use custom icons for each section to provide easy navigation.
- Modern, grid-based layout with ample white space to ensure readability.
- Use GSAP animations for smooth transitions between sections.
